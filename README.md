# WalletConnect Waku TypeScript Demo

这是一个使用 go-waku 集群实现的 WalletConnect v2 协议演示项目，展示了 Web3 应用和钱包之间的连接和签名流程。

## 功能特性

- 🔗 基于 Waku 网络的去中心化通信
- 🔐 端到端加密的消息传输
- 📱 模拟 DApp 和钱包交互
- ✍️ 消息签名和验证
- 🎯 JWT token 认证
- 🌐 Web 界面操作

## 技术栈

- TypeScript
- Waku SDK
- Ethers.js
- Webpack
- Express.js
